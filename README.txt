
# Les Pointures des Fléchettes - PWA

## Lancer localement

Ouvrez le dossier avec un serveur local :

Python :
python -m http.server 8080

Puis ouvrez :
http://localhost:8080

## Hébergement recommandé

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

## Fonctionnalités

- Installation mobile/desktop
- Hors ligne
- Android / iOS / Windows / macOS / Linux
- Sauvegarde locale
- Plein écran
